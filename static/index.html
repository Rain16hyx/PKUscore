IMPORT GUIDE

从北大树洞导入

隐私说明源码只发送到你电脑上运行的 Python 程序，不会离开本机。导入前仍建议检查并删除不希望保留的信息。
  1. 1
    打开成绩页面

    登录北大树洞，进入「成绩」，等待所有学期和课程完全加载。

  2. 2
    打开开发者工具

    Chrome / Edge 按 F12（Mac 可按 ⌥⌘I），选择顶部的「Elements / 元素」。Safari 需先在设置中开启“显示网页开发者功能”。

  3. 3
    复制完整 body

    在元素树找到 <body>，右键选择「Copy → Copy outerHTML / 复制外部 HTML」。请勿使用“查看网页源代码”，其中通常没有已加载的成绩。

  4. 4
    粘贴并导入

    把复制的内容粘贴到下方。导入会追加到现有记录，同名学期会合并。

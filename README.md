PKUscore 用来整理课程成绩，并按照北京大学绩点公式计算课程绩点、学期 GPA 和总 GPA。成绩可以逐门录入，也可以直接从北大树洞成绩页面导入。

程序只在电脑本地运行。课程数据保存在浏览器本地，导入的成绩页面源码不会发送到互联网。

## 主要功能

- 按学期管理课程，实时计算课程绩点、学期 GPA 与总 GPA
- 直接解析北大树洞成绩页面 HTML，省去逐门录入
- 重新导入时覆盖上一批树洞数据，同时保留手工添加的课程
- 支持百分制、P/F 合格制和 IP 进行中状态
- 根据成绩为课程整行着色，方便快速浏览
- 支持浅色与深色主题
- 无需数据库，也不依赖第三方 Python 库

## 开始使用

PKUscore 目前需要电脑上安装 **Python 3.10 或更高版本**。一键启动器会自动完成端口选择、服务启动和浏览器打开，不需要在终端中输入命令。

### macOS

双击项目根目录中的 `PKUscore.app`。

运行期间请保留 PKUscore 应用开启。从程序坞退出应用后，本地服务也会停止。如果首次打开时出现安全提示，请在 Finder 中右键点击 `PKUscore.app`，选择“打开”并再次确认。

### Windows

双击项目根目录中的 `PKUscore-Windows.cmd`。

运行期间请保留弹出的 PKUscore 窗口，关闭窗口或按 `Ctrl+C` 即可停止服务。使用 Python 官方安装程序时，需要勾选 **Add Python to PATH**。

> 一键启动文件需要留在项目根目录中，请不要将它们单独移动到其他位置。

## 导入树洞成绩

1. 登录北大树洞并进入成绩页面，等待课程信息加载完成。
2. 打开浏览器开发者工具，进入「Elements / 元素」面板。
3. 找到 `<body>` 节点，右键选择「Copy → Copy outerHTML」。
4. 在 PKUscore 中点击「导入树洞源码」，粘贴后导入。

页面内也提供了更详细的操作提示。再次导入时，系统会先移除上一次导入的结果，再写入最新数据，不会重复显示课程。

## 计算规则

百分制课程按照北京大学绩点公式换算：

```text
G = 4 - 3 × (100 - S)² / 1600
```

其中 `S` 为百分制原始成绩。

| 成绩类型 | 处理方式 |
| --- | --- |
| 60–100 分 | 按公式计算绩点，并计入 GPA |
| 低于 60 分 | 课程绩点为 0，显示为 F |
| P/F 合格制 | 保留课程和结果，不计入 GPA |
| IP 进行中 | 显示为 IP，不计入 GPA |

学期 GPA 是该学期所有计绩课程按学分加权后的平均值；总 GPA 则对全部计绩课程统一按学分加权。

## 数据与隐私

课程记录保存在浏览器的 `localStorage` 中。后端服务只监听本机地址 `127.0.0.1`，树洞源码和成绩数据不会离开当前电脑。

清除浏览器站点数据或更换浏览器会使本地记录消失。需要长期保留时，请避免清理 PKUscore 对应的浏览器数据。

## 命令行启动

如果更习惯使用终端，也可以运行：

```bash
python3 app.py
```

默认访问地址为 <http://127.0.0.1:8000>。需要使用其他端口时：

```bash
python3 app.py --port 8080
```

按 `Ctrl+C` 即可停止服务。
